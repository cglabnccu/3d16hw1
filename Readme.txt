遊戲想法：
	老師於上課demo過一個歷屆作品，是用3X3的魔法陣，將人物移動到空格，弱無法在時間內移動到空格，則會有人物被燒傷的畫面。
	於是我也想做類似的遊戲，而我採用類似的概念，結合太鼓達人裡面有一個操作，是要在時間結束前連續敲擊鼓面以擊破氣球，
	做出這個遊戲。

遊戲玩法：
	上方有6顆心，當心扣完則遊戲結束。
	遊戲有兩階段：
		-第一階段
			3X3的陣列，由左而右，由下到上，依序對應鍵盤的Z、X、C、A、S、D、Q、W、E
			當3X3的陣列缺少哪一個，就點擊對應的鍵，若點擊失敗則扣一顆心，點擊成功即進入下一階段。
		-第二階段
			依據左上方的人物的手指方向，連續點擊對應的上、左、右鍵，每點擊一次，左下角的圖案就會越變越大，
			當在時間內點擊到對應的次數，即會回到第一階段，並補充1顆心(上限是6顆)。
			若點擊錯誤的方向，每一次會扣一顆心。
			未能達成指定的點擊次數，則扣一顆心。
			左上角會顯示，進入該次第二階段後，已經點擊(hit)的次數。
		隨著遊戲進行，第一階段的空格變換位置速度會越來越快，第二階段的時間也會縮短、需要點擊的次數也會增加，
遊戲目的：
	玩家可藉由這個遊戲，訓練手眼協調能力，以及藉由快速點擊紓解壓力。
